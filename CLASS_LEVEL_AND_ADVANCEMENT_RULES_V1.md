## 核心定位

等級主要作為轉職門檻。

角色主要成長來源不是單純升級，而是：

| 成長來源 | 說明 |
|---|---|
| 轉職 | 開啟新職業定位與成長方向 |
| 技能 | 5轉、6轉後技能欄位完整 |
| 被動 | 職業特色與流派成型 |
| 裝備 | 中後期主要戰力來源 |
| 收藏品 | 全旅團微量永久加成 |
| 古代靈藥 | 單一角色永久微量成長 |
| 隱藏職業 | 特殊路線與玩法變化 |

---

## 核心原則

Lv60 是 CBT 最高轉職門檻。

封測進度主要透過「經驗值需求曲線」控制。

不得以提高轉職等級門檻作為主要延長遊戲時長手段。

## CBT 開放範圍

封測版本開放至：

| 項目 | 規則 |
|---|---|
| 最高轉職 | 6轉 |
| 最高建議等級門檻 | Lv60 |
| 7～11轉 | 不開放 |
| 7～11轉資料 | 可保留為未開放資料 |
| 7～11轉實作 | 不得進入玩家可用流程 |

---

## 封測節奏目標

無課玩家：

約 30～50 天逐步抵達 6轉 / Lv60。

課金玩家：

可比無課快約 15 天左右抵達 6轉 / Lv60。

此為節奏目標，不是硬性每日限制。

---

## 經驗曲線方向

Lv1～30：

前期較快，讓玩家理解基礎職業與分支。

Lv30～45：

中期開始放慢，進入專業職業養成。

Lv45～60：

明顯放慢，作為 CBT 後半段主要養成目標。

本文件不建立經驗值數值表。

## 技能解鎖規則

每位角色完整技能框架為：

| 類型 | 數量 |
|---|---:|
| 普通攻擊 | 1 |
| 主動技能 | 2 |
| 被動技能 | 2 |

---

## 技能解鎖節點

| 轉職階段 | 解鎖內容 | 說明 |
|---|---|---|
| 1～4轉 | 普通攻擊1、主動技能1、被動技能1 | 基礎職業特色 |
| 5轉 | 主動技能2 | 派系技能開始成形 |
| 6轉 | 被動技能2 | CBT職業流派完整成型 |
| 7～11轉 | 不新增第3主動 / 第3被動 | 後續版本以特性、覺醒、數值與職業定位擴充 |

---

## 轉職後等級規則

| 項目 | 規則 |
|---|---|
| 轉職後等級 | 不重置 |
| 轉職後經驗 | 保留 |
| 是否回到Lv1 | 否 |
| 轉職定位 | 等級達標後選擇新職業路線 |

範例：

Lv45 角色完成 5轉後，仍維持 Lv45。

## 轉職消耗規則

目前暫定方向：

| 轉職階段 | 消耗方向 |
|---|---|
| 1～3轉 | 金幣 |
| 4～6轉 | 金幣 + 職業證明 / 專業素材 |
| 7～11轉 | 正式版後續規劃 |
| 隱藏職業 | 隱藏職業解鎖條件 + 對應轉職費用 |

---

## 覺悟 / 停轉規則

4轉後可選擇停留當前職業繼續升級。

不強制進入下一轉。

用途：

| 用途 | 說明 |
|---|---|
| 保留職業外觀 | 玩家可停留喜歡的職業階段 |
| 保留職業定位 | 不強迫角色離開既有流派 |
| 延後轉職決策 | 玩家可等待素材、裝備或路線確認 |

---

## 隱藏職業相容規則

隱藏職業解鎖不等於立即可轉職。

玩家完成隱藏職業條件後：

| 狀態 | 規則 |
|---|---|
| 已解鎖 | 加入帳號職業圖鑑 |
| 未達前置職業 | 不能轉職 |
| 達到前置職業 | 可選擇隱藏職業路線 |
| CBT未開放階段 | 顯示「此職業將於未來版本開放」 |

隱藏職業詳細規則以：

HIDDEN_CLASS_SYSTEM_V2_REVIEW.md

為準。
