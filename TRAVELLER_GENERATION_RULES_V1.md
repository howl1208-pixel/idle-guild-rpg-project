# Traveller Generation Rules v1

## 旅者池基本規則

| 項目 | 規則 |
| --- | --- |
| 酒館顯示旅者數 | 6位 |
| 每次刷新數量 | 2位 |
| 非課玩家刷新週期／<br>課金玩家刷新週期 | 6小時／<br>得以加速成為4小時刷新一次 |
| 刷新淘汰 | 最舊2位旅者離開 |
| 傳聞旅者 | 與一般旅者共用6格 |
| 隱藏職業 | 不進入任何旅者池 |

---

## 轉職階段生成權重

| 轉職階段 | 生成權重 |
| --- | --- |
| 1轉 | 65% |
| 2轉 | 25% |
| 3轉 | 8% |
| 4轉 | 2% |
| 5轉以上 | 0% |

---

## 設計理由

多數職業從3轉開始進入主要派系分流。

因此旅者池不應大量生成高轉職成品角色。

旅者生成系統核心理念：

玩家培養旅者，而不是直接取得完成品。

---

## 普通旅者

| 項目 | 規則 |
| --- | --- |
| 定位 | 一般冒險者 |
| 可生成轉職階段 | 1～4轉 |
| 人格 | 無人格或1個人格 |
| 是否可繼續培養 | 是 |
| 是否可自然刷新 | 是 |

---

## 優秀旅者

| 項目 | 規則 |
| --- | --- |
| 定位 | 有潛力的冒險者 |
| 最大生成轉職階段 | 2轉 |
| 人格 | 主人格1個 + 副人格1個 |
| 是否可繼續培養 | 是 |
| 是否可自然刷新 | 是 |

設計目的：

優秀旅者不是高轉職完成品。

優秀旅者是較好的培養胚子。

---

## 傳聞旅者

| 項目 | 規則 |
| --- | --- |
| 定位 | 有故事與特殊經歷的人物 |
| 最大生成轉職階段 | 4轉 |
| 人格 | 主人格1個 + 副人格1個 |
| 稱號 | 傳聞稱號1個 |
| 是否可繼續培養 | 是 |
| 是否可自然刷新 | 是 |
| 是否使用保底 | 是 |

設計目的：

傳聞旅者可以比普通旅者更成熟。

但不應直接生成5轉以上角色。

---

## 隱藏職業限制

以下隱藏職業不得自然生成：

| 隱藏職業 |
| --- |
| 重劍者 |
| 匿蹤者 |
| 學者 |
| 見習女巫 |
| 小丑 |

---

隱藏職業：

- 不進入普通旅者池
- 不進入優秀旅者池
- 不進入傳聞旅者池
- 不可自然招募

隱藏職業只能依：

HIDDEN_CLASS_SYSTEM_V2_REVIEW.md

規則解鎖。

## 留宿房與旅者池關係

### 旅者池

旅者池代表：

目前正在酒館等待招募的旅者。

---

### 留宿房

留宿房代表：

玩家想暫時保留的人才。

---

### 關係規則

| 項目 | 規則 |
| --- | --- |
| 旅者進入留宿房後 | 從旅者池移除 |
| 留宿中旅者 | 不受6小時刷新淘汰影響 |
| 留宿房用途 | 暫時保留尚未招募的人才 |
| 留宿房是否等於角色倉庫 | 否 |
| 留宿房是否等於現役隊伍 | 否 |

---

## 留宿房容量

| 玩家類型 | 容量 |
| --- | --- |
| 無課玩家 | 2格 |
| 課金玩家 | 5格 |

---

## 留宿房費用

| 旅者類型 | 費用 |
| --- | --- |
| 普通旅者 | 500金幣／天 |
| 優秀旅者 | 1500金幣／天 |
| 傳聞旅者 | 2500金幣／天 |

---

## 生成系統優先順序

旅者生成時應遵守：

1. 隱藏職業限制
2. 傳聞旅者自然出現率與保底
3. 旅者階級規則
4. 轉職階段生成權重
5. 人格與稱號規則
