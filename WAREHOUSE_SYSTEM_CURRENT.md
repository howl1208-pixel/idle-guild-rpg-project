## Consumables Capacity Rule v1

Status: FINAL

正式方向：

消耗品獨立管理。

消耗品不屬於：

- 行囊容量
- 素材容量
- 遠征容量

行囊滿停止拾取僅影響：

- 怪物素材
- 場景素材
- 半成品
- 裝備素材

行囊滿停止拾取不影響：

- 消耗品倉庫
- 特殊收藏倉庫

正式哲學：

容量問題：素材與裝備。

補給問題：消耗品。

兩者獨立管理。

Final Rule：

消耗品不參與容量計算。

消耗品獨立存放。

## Storage Category Rule v1

Status: FINAL

正式分類：

Consumable Storage 包含：

- 藥劑
- 卷軸
- 支援品

Special Storage 包含：

- 寶箱
- 活動箱
- 世界寶箱
- 特殊收藏品

不採用：

- 將寶箱歸類為消耗品。

正式哲學：

治療藥劑與寶箱屬於不同系統。

避免倉庫分類混亂。

Final Rule：

消耗品倉庫：藥劑／卷軸／支援品。

特殊收藏倉庫：寶箱／活動箱／世界寶箱／特殊收藏品。

## 裝備處理規則

| 裝備品質 | 處理方式 |
|---|---|
| 白裝 | 販售 |
| 綠裝 | 販售 |
| 藍裝 | 販售 |
| 紫裝 | 可販售 / 可回收 |
| 暗金 | 僅可回收，不可販售 |

---

## 回收保護規則

| 條件 | 規則 |
|---|---|
| 已裝備物品 | 無法回收 |
| 已鎖定物品 | 無法回收 |
| 紫裝以上 | 預設不進入一鍵回收 |
| 暗金裝 | 預設不進入一鍵回收 |

紫裝與暗金裝若要回收，必須由玩家手動勾選或進入單件回收流程。

---

## 自動販售規則

自動販售只適用於低階裝備。

| 裝備品質 | 自動販售 |
|---|---|
| 白裝 | 可啟用 |
| 綠裝 | 可啟用 |
| 藍裝 | 可啟用 |
| 紫裝 | 不可自動販售 |
| 暗金 | 不可自動販售 |

---

## 一鍵販售規則

| 裝備品質 | 一鍵販售預設 |
|---|---|
| 白裝 | 勾選 |
| 綠裝 | 勾選 |
| 藍裝 | 勾選 |
| 紫裝 | 不勾選，需手動 |
| 暗金 | 不可販售 |

---

## 一鍵回收規則

| 裝備品質 | 一鍵回收預設 |
|---|---|
| 白裝 | 不可回收 |
| 綠裝 | 不可回收 |
| 藍裝 | 不可回收 |
| 紫裝 | 不勾選，需手動 |
| 暗金 | 不勾選，需手動 |

---

## 防誤操作規則

執行以下操作時需二次確認：

- 回收紫裝
- 回收暗金
- 批量處理紫裝
- 批量處理暗金

---

## 設計原則

玩家可以快速清理低階裝備。

玩家不應因自動或一鍵操作失去高價值裝備。

白裝、綠裝、藍裝作為金幣回收來源。

紫裝作為稀有資源回收來源。

暗金作為特殊回收來源，不進入販售系統。
