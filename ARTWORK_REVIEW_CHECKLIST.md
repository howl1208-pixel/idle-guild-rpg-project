# ARTWORK_REVIEW_CHECKLIST.md

Status: Review Draft

Purpose:

Define a consistent review checklist for approving official character artwork.

Every candidate artwork must pass this checklist before entering the Official Art Asset Registry.

---

## Review Categories

### 1. Character Identity

- Matches the Profession Master Reference.
- Profession is immediately recognizable.
- Does not resemble another profession.

### 2. Visual DNA

- Matches the defined Visual DNA.
- Signature Elements are present.
- Visual Boundaries are respected.

### 3. Evolution Stage

- Correct promotion stage.
- Appropriate evolution level.
- Leaves sufficient room for future promotions.

### 4. Silhouette

- Easily recognizable.
- Strong battlefield readability.
- Clear primary silhouette.

### 5. Material Language

- Materials match Character Bible.
- No incorrect material usage.

### 6. Shape Language

- Shape language remains consistent.
- Geometry matches profession identity.

### 7. Production Quality

- No obvious AI artifacts.
- Anatomy is acceptable.
- Equipment is coherent.
- Perspective is stable.

### 8. Future Compatibility

- Suitable as the parent reference for the next promotion.
- Suitable for Battlefield Adaptation.
- Suitable for Sprite Production.

---

## Approval Decision

### PASS

```text
PASS
-> Register in OFFICIAL_ART_ASSET_REGISTRY
-> Become Official Master Reference
```

### FAIL

```text
FAIL
-> Return for revision
-> Do NOT enter Official Registry
```
